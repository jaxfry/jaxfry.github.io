Background: Tamagotchi was a digital pet created in Japan in 1996. Users cared for there tamagotchi by feeding, playing, and cleaning up after them, mimicking owning a real pet! After a while there pet could grow and evolve if it was well taken care of. Tamagotchi was a very popular product all around the globe.

Task Advanced Version: Create your own version of a Tamagotchi in Python with a text based interface! Include attributes such as happiness and hunger, and be able to play with it. Be creative, expand on from there! If you are wanting to expand upon this project, here are some ideas: - Add more attributes such as age, weight, and energy - Add more activities such as feeding, cleaning, and sleeping - Add a GUI interface or ASCII art - Add a save feature - If you're feeling really ambitious, add a multiplayer feature. Fun fact, Tamagotchi did have a multiplayer feature in later versions then the original!

Task Beginner Version: Create a simpler version where users can only feed or play with the pet and see how it effects the pets attributes.
